You are viewing the ACE system status.

Use the `ace_status` tool to get the current status of the ACE learning system, including:
- Whether ACE is enabled or disabled
- Learning mode (async or sync)
- ACE model being used
- Global skillbook statistics
- Per-agent skillbook statistics

**Arguments:** $ARGUMENTS

If a specific agent ID is provided, show stats only for that agent.
Otherwise, show stats for all configured agents.

Execute the ace_status tool now.
