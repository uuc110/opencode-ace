description: Import a skillbook from a file
---

You are importing a skillbook from a file.

Use the `ace_import` tool to load skills from a previously exported file.

**Arguments:** $ARGUMENTS

Parse the arguments to extract:
- agentId: The agent to import skills into (required)
- inputPath: Path to the file to import (required)
- merge: If mentioned, merge with existing skills instead of replacing

Execute the ace_import tool with the parsed arguments.
