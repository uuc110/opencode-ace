description: Inspect skills in an agent's skillbook

You are inspecting skills in an agent's skillbook.

Use the `ace_inspect` tool to view detailed information about learned skills.

**Arguments:** $ARGUMENTS

Parse the arguments to extract:
- agentId: The agent to inspect (required)
- section: Optional section filter
- limit: Optional number of skills to show (default: 10)
- sortBy: Optional sort order (helpful, harmful, recent)

Execute the ace_inspect tool with the parsed arguments.
