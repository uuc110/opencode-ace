description: Export an agent's skillbook to a file

You are exporting an agent's skillbook.

Use the `ace_export` tool to save the skillbook to a file for backup or sharing.

**Arguments:** $ARGUMENTS

Parse the arguments to extract:
- agentId: The agent whose skillbook to export (required)
- outputPath: Optional custom output path

If no output path is provided, the skillbook will be saved to the default exports directory.

Execute the ace_export tool with the parsed arguments.
